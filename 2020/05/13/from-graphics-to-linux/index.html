From Graphics to Linux

2020-05-13 work

下午，我将以初学者的姿态投入到 Linux Kernel 领域。谈起工作的方向，这是我第二次主动切换方向。第一次主动切换方向，是在2020年1月份，申请去做图形学相关的工作。这一次，是从图形学方向切换到 Linux Kernel 方向。

图形学是我非常感兴趣的领域。图形学能够直观地表达代码所做的工作，如绘制一条优美的曲线，着色等。但是，如果想要在图形学领域小有成就，需要具备深厚且扎实的数学功底。而深厚的数学理论，是我所不擅长的。 所以，经过一个月的思想斗争，最终，我决定，去从事另一个感兴趣的领域，Linux Kernel。

当真正接触到 Linux Kernel 的实际开发工作后，我会喜欢其工作内容吗？我不知道。

昨天，Leader WJZ 给了我一些建议：“交付项目和技术项目相比，更有节奏，但是也更加关注时间节点。有可能你从事 Linux Kernel 开发工作之后，也会很受挫。但是，你一定要积极地面对。”

写在方向变换前夕。

遇到困难，咬咬牙，跺跺脚，也就坚持下去了。
